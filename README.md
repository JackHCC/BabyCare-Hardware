## 项目展示
视频地址：[Video](https://www.bilibili.com/video/BV1pE411w71e/)

### 传感器清单：
麦克风声音传感器，震动传感器，人体红外传感器，WIFI模块，交通灯模块，蜂鸣器，温湿度传感器，光敏传感器，摄像头。

两个文件wifi模块分别用http和mqtt协议实现
